Normale Commands

!afk oder !lurk
Stellt deinen Status auf AFK/Lurk und merkt sich, wie lange du weg warst.
Nützter 👉 Jeder Nützter darf Die Command nutzen
!ping
Zeigt dir den Ping und die Latenz des Bots an.
Nützter 👉 Jeder Nützter darf Die Command nutzen
!leistung
Zeigt dir die RAM-Auslastung des Bots an und in wie vielen Kanälen er aktiv ist.
Nützter 👉 Jeder Nützter darf Die Command nutzen
!7tv
Gibt dir den Link zur 7TV Seite, um Emotes sehen zu können.
Nützter 👉 Jeder Nützter darf Die Command nutzen
!chatterino
Gibt dir den Link zur Chatterino/Chatterino Homies Seite.
Nützter 👉 Jeder Nützter darf Die Command nutzen
!bild
Gibt dir Links zu Seiten, wo du Bilder hochladen kannst.
Nützter 👉 Jeder Nützter darf Die Command nutzen
!followbots
Gibt dir ein Link zu einer Seite wo du Follower Bots Removen kannst.
Nützter 👉 Jeder Nützter darf Die Command nutzen
